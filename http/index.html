NodeMCU Platform

This page is served by nodemcu-httpserver variant running on an ESP8266 that uses the NodeMCU firmware. NodeMCU puts a Lua interpreter inside the ESP8266. This is surely one of the smallest web servers to date!
The NodeMCU Platform is a modularization of common NodeMCU tasks. The goal is to allow focus on development of the specific application without having to worry about common things like setting up the wifi, compiling lua files into lc, etc.

Index

  • Wifi Configuration UI: Set wifi parameters for Access Point and Station modes
  • Web Server Configuration UI: Set access control for webserver
  • NodeMCU info: Shows some basic NodeMCU info
  • Server file list: Displays a list of all http server files
  • Garage door opener: Control GPIO lines via the server
